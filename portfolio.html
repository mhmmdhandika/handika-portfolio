About Me

Hai, perkenalkan nama saya Muhamad Handika Sopian. Saya adalah seorang siswa kelas 12 dari SMK Teknikom Cikarang dan saya mengambil jurusan Teknik Komputer Jaringan, karena saya tertarik dengan komputer dan perkembangan teknologi saat ini.

Awal saya mengenal pemrograman komputer adalah karena saya ikut serta dengan kegiatan ekstrakulikuler yang bernama ICT (Information Club Technology). Pada pelatihan tentang pemrograman dasar, saya agak tertarik dengan namanya istilah "pemrograman". Dan saya mengeksplor apa itu pemrograman dan di waktu yang bersamaan pula saya mengumpulkan resource pembelajaran pemrograman secara online.

Pada awal belajar pemrograman, saya langsung tertarik dengan Web Development dan Android Development. Karena spesifikasi tools saya tidak memadai untuk melakukan pengembangan Android Development, maka saya putuskan untuk fokus ke dalam dunia Web Development khususnya dibidang Frontend Web Development.
